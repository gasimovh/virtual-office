# Virtual Office


### How to run

This app requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies both server and the client side

Server:
```sh
$ cd virtual-office
$ npm install 
```

Client:
```sh
$ cd virtual-office/client
$ npm install 
```

Run the server first:
```sh
$ cd virtual-office/
$ npm run dev
```

Then the client :
```sh
$ cd virtual-office/client
$ npm run serve
```
